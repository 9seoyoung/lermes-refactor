# 파일 등록 (글 올리기)
파일을 1개 or 여러개 올리고 업로드버튼 클릭

단일 등록도 객체가 배열에 감싸져서 옴.

`POST` onopco2.iptime.org:940/api/files

  },
  {
    "fileSn": 184,

# 글 불러오기 (파일 목록 불러오기)
포스트 불러오는 요청을 먼저 해서 formUuid를 받아온다.
받아온 formUuid를 이용해서 
`GET` http://onopco2.iptime.org:940/api/files/formUuid/{formUuid} 로 요청 보냄 

예시 `GET` http://onopco2.iptime.org:940/api/files/formUuid/c6bd16d1a14a463da70856ceb3ce9864

# 파일 다운로드
파일 다운로드는 현재 두가지 방식 공존중임
## 파일 SN으로 다운로드
`GET` http://onopco2.iptime.org:940/api/files/id/{fileSn}
해당 파일 다운로드 됨
## 파일 저장명(UUID명)으로 다운로드
`GET` http://onopco2.iptime.org:940/api/files/{storedFileName}
해당 파일 다운로드됨

# 파일 미리보기
파일 미리보기는 현재 UUID명으로만 가능함. 요청하면 파일SN으로도 가능하게 만들어드리겠음
`GET` http://onopco2.iptime.org:940/api/files/{storedFileName}/preview
