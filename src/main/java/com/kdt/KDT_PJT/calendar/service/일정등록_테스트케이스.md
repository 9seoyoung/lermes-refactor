# 1. 일정 등록
`POST` http://onopco2.iptime.org:940/api/calendar

모든 요청 url 같음. 다만 관리자(대표, 직원)의 경우 기수번호(cohortSn) json에 실어보내야함
## 1.1 관리자인 경우 (권한 번호 3번 이하)

### 1.1.1 공식 일정(PRVT_YN=0) 등록 가능 <br>
요청
```json
{
  "cohortSn": 100,
  "eventBgngDt": "2025-09-18T09:00:00",
  "eventEndDt":  "2025-09-20T12:00:00",
  "eventNm":     "관리자 공식일정 등록",
  "rmrkCn":      "공식일정임 이때는 cohortSn필수",
  "prvtYn":      0
}
```
응답
```json
{
    "calSn": 9,
    "cohortSn": 100,
    "eventBgngDt": "2025-09-18T09:00:00",
    "eventEndDt": "2025-09-20T12:00:00",
    "eventNm": "관리자 공식일정 등록",
    "rmrkCn": "공식일정임 이때는 cohortSn필수",
    "userSn": 20,
    "eventRegDt": "2025-09-18T16:30:14",
    "prvtYn": 0
}
```
### 1.1.2 개인 일정(PRVT_YN=1) 등록 가능
요청
```json
{
  "eventBgngDt": "2025-09-18T09:00:00",
  "eventEndDt":  "2025-09-20T12:00:00",
  "eventNm":     "관리자 개인일정",
  "rmrkCn":      "ㅇㅇ",

{
    "calSn": 12,

    "userSn": 20,

}
```

# 2. 일정 조회
## 2.1 관리자인 경우 
### 해당 기수의 해당 날짜의 공식 일정만 뜨는거 (제목이랑 일정SN만 주면 될듯)
### 해당 날짜의 개인일정만 뜨는거 (제목이랑 일정SN만 주면 될듯)

### 해당 기수의 모든 날짜의 공식일정만 뜨는거
### 해당 기수의 모든 날짜의 개인일정만 뜨는거

### 월을 보내주면 그 월의 개인일정, 해당 기수의 공식일정 띄우는거 (일정SN, 제목, 시작일, 종료일)

## 2.2 강사/학생인 경우
### 해당 날짜의 공식 일정만 뜨는거 (제목이랑 일정SN만 주기)
### 해당 날짜의 개인 일정만 뜨는거 (제목이랑 일정SN만 주면 될듯)

### 모든 날짜의 공식 일정만 뜨는거
### 모든 날짜의 개인 일정만 뜨는거

### 월 보내주면 그 월의 개인일정, 공식일정 띄우는거 (일정SN, 제목, 시작일, 종료일)
