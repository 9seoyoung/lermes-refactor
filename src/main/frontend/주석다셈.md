## 📝 TODO 주석 관리 규칙
### 0. TODO 주석 세팅
```
1. IntelliJ Live Template 등록하기

설정 열기(Ctrl + Alt + S (윈도우/리눅스) / Cmd + , (맥))

Live Templates 메뉴로 이동(Editor > Live Templates)
- 새 템플릿 추가
- 오른쪽 + 버튼 클릭 → Live Template 선택
- Abbreviation: td
- Description: "TODO with date & user"
- Template text 입력: // TODO-$DATE$-구서영: $END$


변수 지정 (Edit variables... 버튼 클릭)

DATE → date("yyyy-MM-dd")

적용 범위 지정

아래쪽 Applicable in 에서 Everywhere, java 등 필요한 언어 체크 (안 해두면 코드에서 인식 안 됨 ❌)

저장 후 테스트

코드 파일에서 td 입력 → Tab → 자동완성 나와야 함

2. TODO 필터링 생성
-  Settings > Editor > TODO
-  + 눌러서 새 패턴 추가
    - Name: TODO-DATE-USER
    - TODO-\d{4}-\d{2}-\d{2}-[ㄱ-ㅎ가-힣a-zA-Z0-9]+:.*
```
### 1. 기본 키워드

TODO → 구현해야 할 기능

FIXME → 버그 수정 필요

NOTE → 참고 메모

REVIEW → 코드 리뷰 필요

OPTIMIZE → 성능 개선 필요
```
### 2. 작성 형식
```
[KEYWORD]-[날짜]-[작성자]: [내용]

예시
// TODO-2025-09-12-구팡: 로그인 API 연동
// FIXME-2025-09-12-정원: 회원가입 시 null pointer 발생
// NOTE-2025-09-12-구팡: 이 쿼리는 대량 데이터에서 느려질 수 있음
```
### 3. IntelliJ TODO 탭 활용
```
Alt + 6 (Windows/Linux) 또는 Cmd + 6 (macOS) → TODO 목록 열기

날짜/사용자별 필터링 가능

배포 전 TODO/FIXME 항목은 반드시 처리 or 제거
```
### 4. Live Template 단축어
```
td 입력 후 Tab → 자동으로 오늘 날짜 + 사용자 이름 포함된 TODO 생성

예시 자동 생성 결과:
// TODO-2025-09-12-구팡:
```
### 5. 팀 운영 규칙
```
모든 TODO/FIXME에는 날짜와 작성자 이름을 반드시 포함

PR 올릴 때 TODO/FIXME가 남아 있으면 리뷰어에게 반드시 알리기

최종 배포 전 TODO/FIXME를 전부 처리하거나 제거

NOTE/REVIEW/OPTIMIZE는 참고용으로 유지 가능
```
